Welcome to The Teagasc Advisors Application,


Thank you for taking the time to use my application. It has been a long development process and there is still alot of work to carry
out. I apologise for any bugs you may encounter whilst using the application, but with continued support these issues will be resolved.
The Conduct Assessment Feature is designed to carry out a Wholefarm Stocking Rate and Grassland Stocking Rate
This is required in order to continue with assessing a farmer

On both the Import / Export and the Storage feature offer name select boxes, please double click
in order to see Farmers with stocking rates ( Otherwise farmers cannot be further assessed)
The Import / Export function allows importation and exportation of manure or slurry
impacting the stocking rate

The Storage Feature will record sizes of storage containers and inform the farmer whether enough
storage is available.

The UpdateLSU feature is only available after a farmer has been assessed using the ConductAssessment feature

The UpdateLSU Feature will update the livestock numbers affecting the total nitrates and stocking rates

The View Records feature will only be available after a farmer has been assessed using the
ConductAssessments feature.
